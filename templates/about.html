During the semester, we learned to use various langagues and tools such as Python and Flask (which this app is built on). This site is the culmination of those skills.

The Project

This project was designed to compile data from various sources and put them in one, easy-to-read place. In this case, it's the top-100 rated PS4 games on Metacritic.

The Data

The ratings and games were scraped from Metacritic. The length to compleition data was compiled from HowLongToBeat (HLTB), which is a site that pulls user-submitted data on game completion. The specifc data used was the combined completion time of the main story of the game (if there is one) and any extra content.
